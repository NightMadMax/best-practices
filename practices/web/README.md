# Практики: веб-разработка

Практики веб-разработки: нюансы, типовые ошибки, чек-листы.

## Формат записи

### <Название практики>

- **Контекст:** когда применимо.
- **Проблема:** что шло не так.
- **Решение:** как делать правильно.
- **Источник:** проект, из которого пришёл урок.
- **От кого:** пользователь (имя/роль), автор коммита или агент.
- **Дата:** когда добавлено или подтверждено.
