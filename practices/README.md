# Принятые практики

Каждая принятая практика хранится отдельным файлом
`practices/<stack>/PC-<year>-<number>-<slug>.md` и сохраняет ID исходного
кандидата. README каждого раздела служит индексом, а не контейнером практик.

Формат определён в
[[docs/architecture/decisions/ADR-0003-one-practice-per-file]]; заготовка —
[[practices/_TEMPLATE|_TEMPLATE.md]]. Кандидаты из [[candidates/README]] не
считаются принятыми и потребителям не доставляются.
