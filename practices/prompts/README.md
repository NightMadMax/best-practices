# Практики: промпты и skill-паттерны

Что хорошо сработало в работе с агентами: удачные промпты, формулировки и
skill-паттерны, которые стоит переиспользовать. Общий стандарт правил агента
живёт в `new-project-rules`; здесь — прикладные приёмы из проектов.

## Формат записи

### <Название приёма>

- **Когда применять:** задача/ситуация.
- **Промпт / паттерн:** формулировка или структура (в блоке кода, без секретов).
- **Почему работает:** какой эффект и на чём проверено.
- **Источник:** проект, из которого пришёл приём.
- **От кого:** пользователь (имя/роль), автор коммита или агент.
- **Дата:** когда добавлено или подтверждено.
