# Практики: сниппеты и конфиги

Переиспользуемые куски кода и конфигураций, проверенные на проектах. Только
обезличенные фрагменты — без секретов, токенов и приватных хостов.

## Формат записи

### <Название сниппета>

- **Контекст:** для чего и в каком стеке.
- **Сниппет:** фрагмент в блоке кода (плейсхолдеры вместо реальных значений).
- **Заметки:** ограничения, зависимости, на что заменить плейсхолдеры.
- **Источник:** проект, из которого пришёл сниппет.
- **От кого:** пользователь (имя/роль), автор коммита или агент.
- **Дата:** когда добавлено или подтверждено.
